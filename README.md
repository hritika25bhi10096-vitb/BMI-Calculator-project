Overview
This project is a simple, command-line utility implemented in Python to calculate an individual's Body Mass Index (BMI) and classify their weight status. It adheres to standard WHO guidelines for classification.
The program focuses on clear input/output and direct calculation, making it easy to use for immediate health monitoring.

Features 
The application performs the following core functions:
1.	Data Input: Accepts user input for height (in meters) and weight (in kilograms).
2.	BMI Calculation: Computes the Body Mass Index using the formula: 
                                            BMI = (weight)/(height)**2
3.	Classification: Categorizes the calculated BMI into one of four groups: Underweight, Normal weight, Overweight, or Obese.

How to run
1.	Open terminal or command prompt
2.	Navigate to the folder where main.py is saved
3.	Run the program using python
